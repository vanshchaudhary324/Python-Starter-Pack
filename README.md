Program Suite Collection
This folder contains a unique collection of interactive Python programs designed for fun, utility, and learning. Each program offers a distinct experience, from games to tools, all created on 6/1/2025 at 10:29 AM.
Programs Included

Guess the Number: A number guessing game.
Hangman: A classic word-guessing game.
Madlibs: A fill-in-the-blank story game.
Password Generator: Create secure passwords.
QR Code Generator: Generate QR codes.
Quiz Game: Test your knowledge with quizzes.
Rock Paper Scissors: Play the classic hand game.
Story Generator: Create random stories.

How to Use

Ensure you have Python installed.
Navigate to each folder and run the respective Python script (e.g., python hangman.py).
Follow the on-screen instructions for each program.
